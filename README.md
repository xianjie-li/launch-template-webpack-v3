# webpack多页面配置
[github](https://github.com/qq1073830130)

### 食用说明:
<hr />
基于webpack3，等webpack4坑踩完后会升级...
 
###### 安装依赖
```
npm install or yarn     //使用你喜欢的包管理器安装依赖
```
###### 命令 
```
npm run dev         //开发服务器
npm run build       //构建线上版本
```
###### 功能概览
- <b style="color: red;">多页面</b>
- 自动根据创建的html页面添加webpack入口
- es6+语法
- sass
- postcss
- sourceMap
- 代码压缩
- 精确的处理模块打包后的hash
- vue及react支持
- pug
